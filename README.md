## 👨‍🏫 프로젝트 소개
간단한 메타버스를 구현한 프로젝트입니다.
메타버스답게 멀티플레이가 가능하며
간단한 미니게임을 확인해볼 수 있습니다.

## 💜 주요기능

- 기능 1
    유니티 포톤을 사용하여 실시간으로 유저들과 함께 플레이할 수 있습니다.

- 기능 2
    NPC와 상호작용하여 대화를 할 수 있습니다.
  
- 기능 3
    채팅을 통해 실시간으로 유저들과 소통할 수 있습니다.
  
- 기능 4
    탈것과 스킨을 변경할 수 있습니다.
